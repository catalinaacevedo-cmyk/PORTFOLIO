Habilidades aprendidas durante el módulo 2 del bootcamp.

Sesión 01

  • La primera tarea que realizamos fue crear una página web dedicada a un personaje famoso.
  • A continuación, elaboramos un currículum vitae y aprendimos a utilizar el formato de texto en negritas y cursiva

    • Sesión 02

  • Aprendimos a crear formularios, en los cuales utlizamos elementos de entrada, como Input, Textarea, Select, para captar información.

  • Los formularios utilizan el metodo de envío como GET para obtener datos o POST para captar información.
    • Sesión 03

    Aprendimos CSS y realizamos tablas con colores. También aprendimos a crear botones con estilos.

    Ejemplo de tabla:

    Nombre Edad Ciudad
    Dogma 8 Santiago
    Victoria 33 Dinamarca
    Loreto 34 Copiapo
    Sesión 04

    Aprendimos sobre la responsabilidad de diseño con el enfoque mobile first.

  • Google favorece los sitios que están optimizados para móviles, lo que puede mejorar el posicionamiento en los motores de búsqueda.

  • Esto asegura la funcionalidad, independientemente del dispositivo.

    • Sesión 05

    • Aprendimos sobre Bootstrap y realizamos una tabla. Creamos una página web para una aerolínea llamada SkyP.

    • Aprendimos a usar las NAVBAR y CARDS de bootstrap

    Sesión 06

    • Creamos un formulario utilizando Bootstrap, que incluía un componente llamado jumbotron.

    • Diseñamos una página con un buscador de vuelo para una aerolínea similar a LAN

    Sesión 07

    • Aprendimos JavaScript e hicimos un ejercicio para comparar números con alertas.

    • También creamos un menú utilizando un Prompt y una Alert para una empresa de celulares.

    Sesión 08

    • Aprendimos sobre control de flujo en JavaScript y realizamos un menú que nos permitía salir y elegir distintas opciones.

    • Creamos un menú similar al de un banco en línea.

    Sesión 09

    • Manipulamos el DOM para hacer que los elementos aparecieran y desaparecieran, o se agrandaran mediante JavaScript.

    • Creamos una lista de tareas pendientes, donde podíamos agregar nuevas tareas y eliminarlas.

    Sesión 10

    • Repetimos el ejercicio de la sesión 09, pero esta vez los elementos aparecieron y desaparecieron con jQuery.

    • En este ejercicio, tomamos la página que habíamos realizado anteriormente de SkyP y la actualizamos, añadiendo la opción de agrandar la descripción.

    Sesión 11

    • Aprendimos a utilizar Git Bash. Git es un sistema que controla todas las versiones que existan de nuestro código, permitiéndonos navegar entre las diferentes versiones que han existido, crear ramas, que serán lugares donde podremos trabajar sin alterar la versión estable actual y que, cuando se encuentre terminada una nueva funcionalidad, nos permite incorporarla a la versión estable sin riesgos

    Sesión 12

    • Aprendimos sobre GitHub que es un proveedor de alojamiento de repositorios en la nube (remoto), gestionado por Git. Es el más usado y permite albergar un número ilimitado de repositorios tanto públicos como privados.
